Summer Cleanup

Many days of summer have passed, and today, after taking some time off, I’ve cleaned up the team’s GitHub repositories, “cleaning” up many things. Here I’d like to mention that I’ve removed some repos. Sorry for that, in case you were using them. All the repos have been created freshly, for I see no need for a “android_frameworks_base-cyngn” and “android_frameworks_base-cody”. So, decided to go a lil’ step further, and use named branches+android versions. Most of the repos would now have only one branch, (thank God), named ‘cody40’; (cody: name of ROM, 40: 4.0 android version) thus making it easier for me to maintain the repos.

Now that I’ve finally managed to cleanup all the repos, lemme just do some CodyROM builds and check if they’re working fine :p

Happy Summer Holidays!
